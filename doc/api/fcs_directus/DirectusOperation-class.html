DirectusOperation

DirectusOperation class

Représente une opération Directus.

Les opérations sont des blocs de construction pour les flows. Chaque opération effectue une action spécifique et peut être chaînée avec d'autres opérations pour créer des workflows complexes.

Exemple d'utilisation :

// Une opération est généralement chargée dans le contexte d'un flow
final flow = await client.flows.getFlow(
  'flow-id',
  query: QueryParameters()
    ..fields = ['*', 'operations.*'],
);

final operations = flow.operationsList;
for (final op in operations) {
  print('Operation: ${op.name.value} (${op.type.value})');
}
