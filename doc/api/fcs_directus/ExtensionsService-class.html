ExtensionsService

ExtensionsService class

Service pour gérer les extensions dans Directus.

Directus peut être facilement étendu par l'ajout de plusieurs types d'extensions, y compris les layouts, interfaces et modules.

Note: Les extensions sont généralement en lecture seule via l'API, avec possibilité de mise à jour de la configuration seulement.

Exemple d'utilisation:

// Lister toutes les extensions
final extensions = await client.extensions.getExtensions();

// Mettre à jour la configuration d'une extension
await client.extensions.updateExtension(
  'my-extension',
  {'enabled': true},
);
